Development version

Use git to follow the latest changes or view them on this webpage. This command checks out the 'next' branch:

  $ git clone https://github.com/i3/i3

The 'stable' branch is the current stable version plus bugfixes. The 'next' branch is the next version, yet to be released. Use git checkout stable to switch to the 'stable' branch.

If you prefer to download a tarball, or if you cannot use git for whatever reason, you may download the current stable branch from https://github.com/i3/i3/archive/stable.tar.gz.

Announce mailing list

If you want to be notified when a new version of i3 is released, please subscribe to the announce mailing list.
